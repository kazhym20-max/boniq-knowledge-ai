# BONIQナレッジAI — 使い方ガイド

立川光昭・ジェイ・エイブラハムの175本分の講義をAIが検索して答えてくれるツールです。

---

## これは何？

マーケティング・商品企画・販売戦略について質問すると、  
立川さん・Jayの講義内容をもとにAIが具体的な回答を作ってくれます。

「本を読む時間はないけど、今すぐ答えが欲しい」という場面で使います。

---

## 使い方

ターミナルで以下を実行するだけです。

```bash
cd /Users/hatakazuhiro/Desktop/cursor-master/00_knowledge/rag
python3 query_rag.py "ここに質問を書く"
```

## 検索対象を絞る（オプション）

立川さんだけ、またはJayだけに絞って検索できます。

```bash
# 立川光昭の講義だけから検索
python3 query_rag.py "質問" --source tachikawa

# ジェイ・エイブラハムの講義だけから検索
python3 query_rag.py "質問" --source jay_abraham
```

どちらを使うか迷ったら:
- 商品企画・ヒット商品・PR → 立川さん
- マーケティング理論・LTV・JV・戦略 → Jay

## 回答の読み方

回答の下に「参照ナレッジ」が表示されます。

```
参照ナレッジ:
  [0.54] 立川 / ヒット商品はターゲティングが9割
  [0.51] Jay / 差別化して価格競争から抜け出すには
```

- 数字（0.54など）は関連度。1.0が最高、0.3以上で信頼できる結果
- どの講義をもとに回答したかが分かる

## Claudeに切り替えたい場合

今はGPT-4o-miniで動いています。  
Claude Sonnetに切り替えるには:

1. https://console.anthropic.com/settings/keys でAPIキーを取得
2. `.env` ファイルを開いて `ANTHROPIC_API_KEY=` の後に貼り付け
3. 次回から自動でClaudeで回答（変更不要）
